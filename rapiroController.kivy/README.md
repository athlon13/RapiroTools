### client setup
#### Android 
    * instal Kivy Launcher
        * https://play.google.com/store/apps/details?id=org.kivy.pygame
    
    *create folder
        * /sdcard/kivy/rapiro

    *copy scripts
        * copy client scripts to /sdcard/kivy/rapiro

    run Kivy Launcher and tap rapiro
